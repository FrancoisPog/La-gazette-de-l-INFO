Un mouchard dans un corrigé de Langages du Web

En ce début de semaine, un corrigé frauduleux a été diffusé sur le Moodle du cours de Langages du Web. S'il pouvait à première vue sembler s'agir de pages web tout à fait classiques, la réalité est toute autre. Le code contenait un très discret logiciel espion destiné à dérober les données les plus sensibles des étudiants.
